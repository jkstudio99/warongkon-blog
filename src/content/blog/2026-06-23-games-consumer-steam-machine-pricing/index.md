ข่าว **Games / Consumer** สำหรับรอบวันที่ **23 มิถุนายน 2026** คือ **Valve** เปิดราคา **Steam Machine** รุ่นใหม่อย่างเป็นทางการ และตัวเลขเริ่มต้นทำให้ตลาดเกมคอนโซลต้องคิดหนัก: รุ่น 512GB แบบไม่มี controller เริ่มที่ **1,049 ดอลลาร์สหรัฐ**

Shacknews รายงานวันที่ **22 มิถุนายน 2026** ว่าราคา Steam Machine แบ่งเป็น 4 ชุดหลัก:

- Steam Machine 512GB: **$1,049**
- Steam Machine 512GB + Steam Controller: **$1,128**
- Steam Machine 2TB: **$1,349**
- Steam Machine 2TB + Steam Controller: **$1,428**

ในวันเดียวกัน The Verge รายงานว่า Valve เปิดระบบ reservation ให้ผู้ใช้ลงทะเบียนความสนใจ และจะใช้การสุ่มเพื่อจัดลำดับสิทธิ์ซื้อหลังปิดรอบลงทะเบียนวันที่ **25 มิถุนายน 2026 เวลา 10:00 น. PT / 13:00 น. ET** โดยอีเมลเชิญซื้อชุดแรกจะเริ่มออกในช่วงปลายเดือนมิถุนายน

## Steam Machine ไม่ใช่ console ราคาถูก

ข่าวนี้สำคัญเพราะ Valve ไม่ได้ตั้งราคาแบบ console maker ดั้งเดิม

Sony, Microsoft และ Nintendo มักใช้ hardware เป็นทางเข้าระบบนิเวศ แล้วค่อยทำเงินจาก software, subscription, storefront และ service ภายหลัง แต่ Valve เลือกขาย Steam Machine ในกรอบใกล้เคียง PC component cost มากกว่า

The Verge รายงานว่า Valve ไม่ต้องการ subsidize hardware แบบระบบปิด เพราะมองว่า PC gaming ควรเป็น ecosystem เปิดที่ผู้เล่นเลือก hardware และ software ได้มากกว่า

ผลคือ Steam Machine กลายเป็นของที่อยู่กึ่งกลางระหว่าง console และ PC:

- ใช้ SteamOS เพื่อให้ experience ใกล้ console
- เล่น library บน Steam ได้
- เป็น Linux PC ที่ปรับแต่งได้มากกว่า console
- แต่ราคาเริ่มสูงกว่า console หลักในตลาดอย่างชัดเจน

นี่ไม่ใช่เครื่องสำหรับทุกคนตั้งแต่วันแรก แต่มันเป็นเดิมพันว่า "console-like PC" มี demand จริงแค่ไหนในยุคที่ราคา hardware สูงขึ้น

## ทำไมราคา $1,049 ถึงเป็นสัญญาณใหญ่

สำหรับ consumer gaming ราคาเป็นเรื่องสำคัญมาก เพราะ console สำเร็จได้จาก balance ระหว่าง performance, simplicity และ price

Steam Machine รุ่นใหม่มีข้อดีชัดเจนสำหรับคนที่มี Steam library อยู่แล้ว ไม่ต้องซื้อเกมใหม่ใน ecosystem แยก และสามารถใช้งานเหมือน PC ได้มากกว่า console ปิด

แต่ถ้ามองจากผู้ใช้ทั่วไป ตัวเลข $1,049 ก่อน controller ทำให้คำถามเปลี่ยนทันที:

- ทำไมไม่ซื้อ PS5, Xbox หรือ Switch 2
- ทำไมไม่ประกอบ PC เอง
- SteamOS และความสะดวกบนทีวีคุ้มส่วนต่างหรือไม่
- เกมที่ต้องการเล่นรองรับ Linux/Proton ดีพอหรือยัง
- อายุการใช้งานและ upgrade path จะต่างจาก console อย่างไร

ราคานี้จึงไม่ได้วัดแค่ความน่าสนใจของเครื่อง แต่วัดว่าผู้บริโภคพร้อมจ่าย premium ให้ openness และ Steam library มากแค่ไหน

## Reservation system สะท้อนปัญหา supply และ scalping

Valve ใช้ระบบลงทะเบียนและสุ่มสิทธิ์ซื้อ ซึ่งเป็นสัญญาณว่าบริษัทคาดว่า demand ช่วงแรกอาจสูงกว่าสินค้าที่มี หรืออย่างน้อยต้องการลดปัญหา bot กับ scalper

The Verge รายงานว่าผู้ใช้ต้องมีบัญชี Steam ที่อยู่ในสถานะดี และมีเงื่อนไขการซื้อก่อนวันที่ **27 เมษายน 2026** เพื่อให้สิทธิ์ซื้อไม่ถูกครอบโดยบัญชีใหม่ที่สร้างมาเพื่อเก็งกำไร

นี่เป็นบทเรียนจากสินค้า hardware รุ่นก่อน ๆ ของ Valve ที่ขายหมดเร็วมาก โดยเฉพาะ Steam Deck และ Steam Controller รุ่นใหม่

ในมุมผู้บริโภค ระบบนี้ดีขึ้นกว่าการเปิดขายแบบ first-come-first-served ที่ bot ได้เปรียบ แต่ก็ยังทำให้ early adopter ต้องรอผลสุ่มและไม่มี guarantee ว่าจะซื้อได้ทันที

## Console market กำลังแพงขึ้นทั้งแผง

Steam Machine ไม่ได้เกิดในสุญญากาศ ปี 2026 เป็นปีที่ hardware gaming เจอแรงกดจากต้นทุน memory, storage, logistics และ component supply

The Verge รายงานแยกอีกชิ้นว่า Valve พูดถึงการเจรจา RAM ในปี 2026 ว่ายากและผันผวนมาก จนส่งผลต่อราคาและ supply ของ Steam Machine โดยตรง

ภาพใหญ่คือ console และ PC gaming กำลังเจอกำแพงต้นทุนพร้อมกัน:

- RAM และ storage แพงขึ้น
- hardware generation ใหม่ต้องใช้ bandwidth สูงขึ้น
- ผู้ผลิตลังเลที่จะ subsidize หนักเหมือนอดีต
- เกม AAA ต้องการ performance สูงกว่าเดิม
- ผู้เล่นคาดหวัง frame rate, resolution และ load time ที่ดีขึ้น

ถ้า trend นี้ต่อเนื่อง เครื่องเกมราคาประหยัดอาจเหลือพื้นที่น้อยลง และตลาดอาจแบ่งชัดขึ้นระหว่าง mass-market portable/console กับ premium PC-console hybrid

## มุมมองของผม

ผมมองว่า Steam Machine รุ่นนี้เป็น product ที่กล้ากว่าที่หลายคนคาด เพราะ Valve ไม่ได้พยายามปลอมตัวเป็น console ถูก แต่ยอมบอกตรง ๆ ว่านี่คือ PC สำหรับห้องนั่งเล่นที่ขาย openness, Steam library และ SteamOS integration

ข้อดีคือมันซื่อสัตย์กับ positioning ข้อเสียคือราคาทำให้ user base ช่วงแรกแคบมาก

ถ้า Valve ทำให้ SteamOS บน desktop แข็งแรงขึ้น และทำให้คนประกอบเครื่องเองใช้ experience แบบ Steam Machine ได้ง่าย ข่าวนี้อาจมีผลใหญ่กว่าตัว hardware รุ่นแรก เพราะมันช่วยผลัก PC gaming ไปสู่รูปแบบ console-like ที่เปิดกว่าเดิม

แต่ถ้า software ยังไม่เนียนหรือ game compatibility ยังมี friction ราคา $1,049 จะทำให้ผู้เล่นทั่วไปตัดสินใจยากมาก

สรุปสั้น ๆ:
**วันที่ 22 มิถุนายน 2026 Valve เปิดราคา Steam Machine เริ่มที่ $1,049 และใช้ระบบสุ่มสิทธิ์ซื้อก่อนปิดลงทะเบียนวันที่ 25 มิถุนายน ข่าวนี้ทำให้เห็นว่าอนาคตของเครื่องเกมอาจไม่ได้มีแค่ console subsidized ราคาถูก แต่รวมถึง PC-console hybrid ราคา premium ที่ขายความเปิดและ library เดิมของผู้เล่น**
