สำหรับหมวด Hardware / Infrastructure ในรอบนี้ ข่าวที่ผมมองว่าสำคัญมากคือการที่ **Anthropic ประกาศเมื่อวันที่ 20 เมษายน 2026 ว่าได้ขยายความร่วมมือกับ Amazon เพื่อสำรองกำลังประมวลผลสูงสุด 5 กิกะวัตต์สำหรับการฝึกและให้บริการ Claude**

นี่ไม่ใช่แค่ข่าว partnership ทั่วไป และก็ไม่ใช่แค่การเพิ่มงบลงทุนเชิงภาพลักษณ์ เพราะในประกาศเดียวกัน Anthropic ระบุรายละเอียดที่มีน้ำหนักมากมาก เช่น

- ใช้งาน **Trainium2 มากกว่าหนึ่งล้านชิป** อยู่แล้ว
- จะมี capacity ใหม่เข้ามาในปี 2026 ทั้ง **Trainium2 และ Trainium3**
- มีแผนใช้ชิปไปถึงรุ่น **Trainium4**
- และขยาย inference ใน **เอเชียกับยุโรป**

สำหรับคนที่ติดตามโครงสร้างพื้นฐาน AI ข่าวนี้สำคัญเพราะมันกำลังบอกว่า **การแข่งขันยุคถัดไปไม่ได้วัดแค่ใครมีโมเดลเก่งหรือเงินเยอะ แต่ใครล็อกห่วงโซ่ compute ได้ลึกกว่า ตั้งแต่ไฟฟ้า ชิป ไปจนถึง distribution ผ่าน cloud**

## เกิดอะไรขึ้นในวันที่ 20 เมษายน 2026

Anthropic ระบุชัดในประกาศวันที่ **20 เมษายน 2026** ว่าบริษัทได้ลงนามข้อตกลงใหม่กับ Amazon เพื่อขยายความร่วมมือเดิม และ secure capacity สำหรับฝึกและ deploy Claude ได้สูงสุด **5 กิกะวัตต์**

นอกจากตัวเลข 5 GW แล้ว ยังมีรายละเอียดที่ควรจับตาอีกหลายจุด

- Anthropic บอกว่าจะ commit มากกว่า **100 พันล้านดอลลาร์** ตลอด 10 ปีข้างหน้าให้กับเทคโนโลยีของ AWS
- capacity ใหม่บางส่วนจะเริ่มเข้ามาในระยะใกล้มาก โดยบริษัทระบุว่าจะมี compute ที่มีนัยสำคัญเพิ่มในช่วงไม่กี่เดือนถัดจากประกาศ
- ข้อตกลงครอบคลุม custom silicon หลายรุ่น ตั้งแต่ **Graviton, Trainium2, Trainium3 ไปจนถึง Trainium4**
- Amazon ประกาศลงทุนเพิ่มใน Anthropic **5 พันล้านดอลลาร์ทันที** และอาจเพิ่มได้อีกสูงสุด **20 พันล้านดอลลาร์**

นี่ทำให้ดีลนี้มีลักษณะเป็นทั้ง

- สัญญาเชิงโครงสร้างพื้นฐาน
- สัญญาเชิงช่องทางการขาย
- และสัญญาเชิงยุทธศาสตร์ระยะยาว

พร้อมกัน

## ทำไมตัวเลข 5 กิกะวัตต์จึงใหญ่กว่าการเพิ่ม GPU cluster ปกติ

ในโลก AI infrastructure ช่วงก่อนหน้า คนชอบพูดกันที่จำนวน GPU หรือชิป แต่พอระบบโตถึงระดับ frontier model จริง ๆ คำถามจะใหญ่กว่านั้นมาก เช่น

- จะหาพลังงานอย่างไร
- จะล็อก supply ชิปได้หรือไม่
- จะวาง workload บน custom silicon ได้ลึกแค่ไหน
- และจะส่ง model ให้ลูกค้าทั่วโลกผ่าน cloud layer ไหน

ตัวเลข **5 GW** จึงมีความหมายมาก เพราะมันบอกว่าการเจรจาในระดับนี้ไม่ใช่การเช่าเครื่องเพิ่ม แต่เป็นการล็อกสิทธิ์เหนือ **ทรัพยากรเชิงระบบ** ที่มีข้อจำกัดจริง

เมื่อ Anthropic ระบุว่าใช้ **มากกว่าหนึ่งล้าน Trainium2 chips** อยู่แล้ว ข่าวนี้ก็ยิ่งตอกย้ำว่าชิปเฉพาะทางของ hyperscaler ไม่ได้เป็นแค่ทางเลือกต้นทุนต่ำอีกต่อไป แต่กำลังกลายเป็นแกนหลักของการขยาย frontier model

## จุดที่น่าสนใจมากคือ AWS ไม่ได้ขายแค่ infra แต่ขาย “full Claude entry point”

อีกประเด็นที่คนอาจมองข้ามคือ Anthropic ระบุว่า **Claude Platform on AWS** กำลังจะมา และลูกค้าจะเข้าถึงฟีเจอร์ของแพลตฟอร์ม Claude ได้ในบัญชี AWS เดิม ภายใต้ control และ billing เดิม

รายละเอียดนี้สำคัญมากในเชิง go-to-market

เพราะมันช่วยลด friction ให้ลูกค้าองค์กรอย่างชัดเจน

- ไม่ต้องเปิดสัญญาเพิ่มหลายชุด
- ไม่ต้องสร้างระบบ governance ใหม่ทั้งก้อน
- และไม่ต้องแยก procurement ระหว่าง model vendor กับ cloud vendor มากเกินไป

ถ้าแปลเป็นภาษาธุรกิจ นี่คือการทำให้ Anthropic ขายได้ลึกขึ้นในองค์กรที่ AWS เป็นมาตรฐานอยู่แล้ว

ดังนั้นข่าวนี้ไม่ได้มีแค่ layer ฮาร์ดแวร์ แต่ยังมี layer distribution ที่แข็งแรงมากซ่อนอยู่

## การขยาย inference ในเอเชียและยุโรปสะท้อนอะไร

Anthropic ระบุด้วยว่าจะขยาย inference capacity ใน **เอเชียและยุโรป** เพื่อรองรับลูกค้าต่างประเทศที่เพิ่มขึ้น

จุดนี้มีผลต่อภาพใหญ่ 2 เรื่อง

เรื่องแรกคือ latency และความพร้อมใช้งานในภูมิภาคเริ่มกลายเป็น competitive issue จริง ไม่ใช่แค่เรื่องเสริม

เรื่องที่สองคือเมื่อดีมานด์นอกสหรัฐโตขึ้น โครงสร้างพื้นฐาน AI ก็ต้องกระจายตามทั้งด้าน

- region
- regulation
- และ data residency

นั่นหมายความว่า data center, chip supply และ cloud region จะยิ่งเชื่อมกันแน่นกว่าเดิม

## ข่าวนี้บอกอะไรเกี่ยวกับตลาด Hardware / Infrastructure ในปี 2026

ผมคิดว่ามีอย่างน้อย 3 สัญญาณสำคัญ

### 1. custom silicon ผ่าน hyperscaler กำลังขึ้นมาเป็นแกนหลัก

การที่ Anthropic commit ระยะยาวกับ Trainium หลายรุ่น บอกชัดว่าผู้เล่น frontier ไม่ได้พึ่งสถาปัตยกรรมเดียวอีกต่อไป และเริ่ม optimize stack กับชิปของ cloud partner อย่างจริงจัง

### 2. compute strategy ตอนนี้ต้องรวม power planning เข้าไปด้วย

การพูดถึงกิกะวัตต์ตรง ๆ คือการยอมรับว่า AI infra ระดับบนสุดหนีข้อจำกัดด้านพลังงานไม่พ้นแล้ว

### 3. โมเดลธุรกิจของ AI vendor เริ่มผูกกับ cloud channel ลึกขึ้น

ถ้าแพลตฟอร์มใช้งานได้โดยตรงใน AWS พร้อม account, control และ billing เดิม การแข่งขันจะไม่ใช่แค่ model-vs-model แต่กลายเป็น **ecosystem-vs-ecosystem**

## มุมมองของผม

ผมคิดว่าประกาศวันที่ **20 เมษายน 2026** นี้สำคัญมาก เพราะมันเผยให้เห็นว่า AI infrastructure รอบใหม่ไม่ได้ขับเคลื่อนด้วยคำว่า “GPU shortage” แบบเดิมอย่างเดียวอีกแล้ว

ตอนนี้เกมใหญ่ถูกตัดสินพร้อมกันใน 4 ชั้น

- พลังงาน
- custom silicon
- cloud region
- และ enterprise distribution

ถ้าจะสรุปให้สั้นที่สุด:
**ข่าวนี้ไม่ได้บอกแค่ว่า Anthropic มี compute เพิ่ม แต่บอกว่าในปี 2026 ผู้ชนะด้าน AI infrastructure ต้องควบคุมเส้นทางจากไฟฟ้าไปถึงลูกค้าองค์กรให้ได้ทั้งเส้น**

และดีล Anthropic-Amazon คือหนึ่งในตัวอย่างที่ชัดที่สุดของการเปลี่ยนผ่านนั้น
