ถ้าจะเลือกข่าวหมวด Hardware / Infrastructure ที่สะท้อนแรงกดดันของตลาด AI ได้ชัดมากในสัปดาห์นี้ ผมเลือก **ผลประกอบการ Microsoft วันที่ 29 เมษายน 2026** เพราะมันแสดงให้เห็นพร้อมกันทั้งสองด้านของสงครามรอบนี้:

- ฝั่งรายได้คลาวด์ยังโตแรง
- ฝั่งต้นทุนและการลงทุนโครงสร้างพื้นฐานก็ยังพุ่งต่อ

ในประกาศอย่างเป็นทางการ Microsoft ระบุว่าไตรมาสสิ้นสุดวันที่ **31 มีนาคม 2026** บริษัทมีรายได้รวม **82.9 พันล้านดอลลาร์ เพิ่มขึ้น 18%** ขณะที่ **Microsoft Cloud มีรายได้ 54.5 พันล้านดอลลาร์ เพิ่มขึ้น 29%** และ **Azure กับบริการคลาวด์อื่นโต 40%**

ตัวเลขเหล่านี้แข็งแรงอยู่แล้ว แต่ประเด็นที่ทำให้ข่าวนี้สำคัญมากขึ้นคือ Reuters รายงานในวันเดียวกันว่า Microsoft วางแผนใช้ **capital spending ตลอดปี 2026 ราว 190 พันล้านดอลลาร์** ซึ่งสูงกว่าที่นักวิเคราะห์คาดไว้มาก และ CFO อย่าง Amy Hood ระบุว่าราว **25 พันล้านดอลลาร์** มาจากต้นทุนชิ้นส่วน เช่น ชิป ที่สูงขึ้น

## ทำไมตัวเลข Azure 40% ยังสำคัญมาก

Azure โต **40%** ในไตรมาสล่าสุด เทียบกับ **39%** ในไตรมาสก่อนหน้า ซึ่ง Reuters ชี้ว่าเป็นการเร่งขึ้นเพียงเล็กน้อย แต่ยังพอช่วยยืนยันได้ว่า demand ฝั่ง AI บนคลาวด์ยังไม่อ่อนแรง

ความหมายของตัวเลขนี้ไม่ได้อยู่แค่ “โตเท่าไร” แต่อยู่ที่การพิสูจน์ว่า Microsoft ยังพอแปลงการอัดเงินเข้า AI infra ให้กลายเป็นรายได้ที่ตลาดยอมรับได้

Microsoft ยังเปิดเผยอีกว่า:

- Intelligent Cloud revenue อยู่ที่ **34.7 พันล้านดอลลาร์**
- commercial remaining performance obligation เพิ่มขึ้น **99% เป็น 627 พันล้านดอลลาร์**

ตรงนี้บอกว่า backlog และงานที่รอรับรู้รายได้ยังหนาแน่นมาก ซึ่งเป็นข้อมูลสำคัญสำหรับคนที่พยายามประเมินว่า capex ก้อนยักษ์เหล่านี้มีโอกาสถูกดูดซับด้วยรายได้ในอนาคตหรือไม่

## จุดที่ใหญ่กว่าผลประกอบการคือ “งบลงทุน 190 พันล้านดอลลาร์”

ผมคิดว่านี่คือแกนหลักของข่าว

Reuters ระบุว่าแผนใช้เงินลงทุนปี 2026 ของ Microsoft แตะ **190 พันล้านดอลลาร์** และยังคาดว่าไตรมาสปัจจุบันจะมี capital spending ราว **40 พันล้านดอลลาร์** ด้วย

ข่าวนี้จึงไม่ได้เป็นแค่ earnings beat หรือ miss แบบรายไตรมาส แต่มันสะท้อนว่า Microsoft เชื่อว่าศึก AI infrastructure ยังอยู่ในช่วงที่ต้องเร่งขยาย capacity ต่อ ไม่ใช่ช่วงแตะเบรก

ถ้ามองในภาษาคนทำระบบ นี่หมายถึงการลงทุนต่อเนื่องในทั้ง stack เช่น

- data center
- server fleet
- accelerator และชิป
- networking
- power และ cooling
- software layer สำหรับ serving workload ขนาดใหญ่

และเมื่อ CFO พูดตรง ๆ ว่ามี **25 พันล้านดอลลาร์** ที่มาจากต้นทุนชิ้นส่วนสูงขึ้น ก็ยิ่งทำให้เห็นว่าการแข่งกันสร้าง infra ตอนนี้ไม่ได้วัดแค่ demand แต่ยังวัดความสามารถในการรับแรงกดดันของ supply chain ด้วย

## ศึกนี้ไม่ใช่แค่เรื่องรายได้ แต่คือเรื่องความเชื่อของตลาด

Reuters รายงานด้วยว่า หลังประกาศผล นักลงทุนยังค่อนข้างจับตาอย่างระวัง เพราะแม้ cloud จะโต แต่การแข่งขันกับ Google และ Amazon ก็รุนแรงขึ้น และคำถามเดิมยังไม่หายไป:

- เงินลงทุนระดับนี้จะคืนกลับมาเร็วพอไหม
- Copilot และบริการ AI ต่าง ๆ จะขยายเป็นธุรกิจใหญ่ได้แค่ไหน
- Microsoft ยังมี edge จาก OpenAI มากพอหรือไม่

น่าสนใจตรงที่ Reuters ยังชี้ว่าข้อตกลงใหม่กับ OpenAI ทำให้ Microsoft ไม่มีสิทธิ์ขายต่อผลิตภัณฑ์ OpenAI แบบ exclusive บนคลาวด์อีกต่อไป นั่นหมายความว่าแม้ Microsoft ยังได้ประโยชน์จากความสัมพันธ์นี้มาก แต่เกมก็เริ่มเปิดกว้างขึ้นสำหรับผู้เล่นรายอื่น

## สิ่งที่คนสาย infra ควรจับตา

สำหรับคนทำแพลตฟอร์มและโครงสร้างพื้นฐาน ข่าวนี้ให้สัญญาณอย่างน้อย 3 ข้อ

### 1. Capacity race ยังไม่จบ

เมื่อ Azure โต 40% และ Microsoft ยังเพิ่มแผนลงทุนขึ้นไปถึง 190 พันล้านดอลลาร์ แปลว่าบริษัทไม่ได้มองว่าปัญหาความต้องการ compute ใกล้จะคลี่คลาย

### 2. ต้นทุนฮาร์ดแวร์กลายเป็นตัวแปรยุทธศาสตร์

ถ้าต้นทุนชิปและคอมโพเนนต์ดันงบขึ้นได้ถึง 25 พันล้านดอลลาร์ แปลว่า supply chain ไม่ได้เป็นแค่เรื่องจัดซื้อ แต่เป็นตัวกำหนดความเร็วในการขยาย infra โดยตรง

### 3. ตลาดเริ่มวัดผลจาก conversion เป็นรายได้จริง

ยุคที่พูดแค่ “เราลงทุน AI เยอะ” เริ่มไม่พอแล้ว นักลงทุนต้องการเห็นว่าคลาวด์โต, backlog โต, usage โต และท้ายที่สุด margin ต้องไม่เสียรูปจนเกินไป

## มุมมองของผม

ผลประกอบการ Microsoft วันที่ **29 เมษายน 2026** ควรถูกอ่านว่าเป็นข่าวของ “เศรษฐศาสตร์โครงสร้างพื้นฐาน AI” มากพอ ๆ กับข่าวของคลาวด์

เพราะมันยืนยันสองเรื่องพร้อมกัน:

- demand ของ AI บนคลาวด์ยังแรง
- แต่ต้นทุนของการแย่งชิงความเป็นผู้นำก็แรงไม่แพ้กัน

ถ้าจะสรุปให้สั้นที่สุด:
**Azure 40% คือหลักฐานว่ารายได้ยังไหล แต่ capex 190 พันล้านดอลลาร์คือคำเตือนว่าศึก AI infrastructure ในปี 2026 ยังอยู่ในช่วงเผาเงินและเร่งขยายกำลังผลิตแบบเต็มตัว**
