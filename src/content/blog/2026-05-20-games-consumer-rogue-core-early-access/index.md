ข่าว **Games / Consumer** สำหรับรอบวันที่ **20 พฤษภาคม 2026** คือการที่ **Deep Rock Galactic: Rogue Core** เข้าสู่ **Early Access** บน PC ผ่าน Steam ในวันที่ **20 พฤษภาคม 2026**

Gematsu รายงานตั้งแต่วันที่ **12 มีนาคม 2026** ว่า Ghost Ship Publishing และ Ghost Ship Games กำหนดวัน Early Access เป็นวันที่ 20 พฤษภาคม และเมื่อถึงวันรันบทความนี้ ข่าวจึงกลายเป็น launch-day signal ที่ชัดสำหรับสาย co-op shooter

ผมหยิบข่าวนี้เพราะ Rogue Core ไม่ใช่แค่ DLC หรือภาคเสริมเล็ก ๆ ของ Deep Rock Galactic แต่เป็นเกม standalone ที่เอาแกน co-op mining shooter ไปออกแบบใหม่ด้วยโครงสร้าง roguelite

## Rogue Core คืออะไร

**Deep Rock Galactic: Rogue Core** เป็น cooperative roguelite first-person shooter ที่อยู่ในจักรวาลเดียวกับ Deep Rock Galactic

ตาม overview ที่ Gematsu อ้างจาก Ghost Ship Publishing ฉากของเกมเริ่มจากเหตุการณ์ **Grayout** บน **Hoxxes IV** ทำให้เหมืองจำนวนมากถูกตัดขาดจากโลกภายนอก บริษัท Deep Rock Galactic จึงส่งทีม **Reclaimers** เข้าไปกู้คืนพื้นที่

จุดที่แตกต่างจากเกมหลักคือผู้เล่นไม่ได้เข้ามาพร้อม build ที่คุ้นเคยแบบเดิม แต่เริ่มแต่ละ run ด้วยอุปกรณ์พื้นฐาน จากนั้นค่อยอัปเกรด gear, weapon และ ability ระหว่างทาง

class ที่ประกาศไว้มี 5 แบบ:

- Guardian
- Spotter
- Falconer
- Slicer
- Retcon

นี่ทำให้ Rogue Core ดูใกล้กับเกม run-based co-op มากกว่า mission-based co-op แบบดั้งเดิม

## ทำไม Early Access สำคัญกับเกมนี้

Deep Rock Galactic เป็นตัวอย่างที่ดีของเกม co-op ที่เติบโตด้วย community feedback, live updates และการจูน balance ระยะยาว

การที่ Rogue Core เลือก Early Access จึงสมเหตุสมผล เพราะ roguelite ต้องพึ่ง loop ที่ละเอียดมาก:

- แต่ละ run ต้องยาวพอดี
- upgrade ต้องมีทางเลือกจริง ไม่ใช่ทางเลือกหลอก
- class ต้องมี identity ชัด
- ความยากต้องกดดันแต่ไม่สุ่มจนรู้สึกไม่แฟร์
- co-op ต้องทำให้ผู้เล่นช่วยกัน ไม่ใช่แค่เล่นข้างกัน

เกมประเภทนี้มักต้องใช้ community จริงในการหา breakpoints ที่ design team มองไม่เห็นจาก internal test

ถ้า Ghost Ship จับ feedback ได้ดี Early Access อาจช่วยให้ Rogue Core สร้างฐานแฟนได้เร็วเหมือนเกมหลักเคยทำไว้ แต่ถ้า loop ยังไม่แน่น เกมก็จะถูกเทียบกับ Deep Rock Galactic ทันที

## Consumer signal: แฟรนไชส์เกม co-op กำลังแตกแขนงอย่างระวัง

ในมุมตลาด consumer Rogue Core เป็นตัวอย่างที่น่าสนใจของการขยายแฟรนไชส์โดยไม่ทำภาคต่อแบบเส้นตรง

แทนที่จะทำ Deep Rock Galactic 2 ตรง ๆ Ghost Ship เลือกแตกเป็น standalone spin-off ที่ยังใช้โลกเดิม น้ำเสียงเดิม และ co-op DNA เดิม แต่เปลี่ยน progression model

ข้อดีคือ:

- แฟนเดิมเข้าใจโลกและ tone อยู่แล้ว
- ทีมพัฒนาสามารถทดลอง roguelite loop ได้โดยไม่ทำลายเกมหลัก
- Early Access ช่วยแยก expectation ระหว่างเกมทดลองกับเกม 1.0
- Steam เป็นช่องทางที่เหมาะกับ community-driven iteration

ข้อเสียคือความคาดหวังสูงมาก เพราะผู้เล่น Deep Rock Galactic มีภาพจำชัดว่าเกมหลักควร "รู้สึก" อย่างไร

## สิ่งที่ควรจับตาหลัง launch day

หลังวันที่ 20 พฤษภาคม 2026 สิ่งที่น่าดูไม่ใช่แค่ยอดผู้เล่นวันแรก แต่คือสัญญาณ retention ในช่วง 2 ถึง 4 สัปดาห์แรก

เกม roguelite co-op จะผ่านหรือไม่ผ่านขึ้นกับคำถามแบบนี้:

- ผู้เล่นอยากเริ่ม run ใหม่ทันทีหรือไม่
- class ต่างกันพอให้ทีมวาง role ได้จริงหรือไม่
- enemy และ biome มีความหลากหลายพอหรือยัง
- progression ระยะยาวทำให้รู้สึกอยากกลับมาเล่นหรือไม่
- community รู้สึกว่า Early Access roadmap น่าเชื่อถือหรือไม่

ถ้าคำตอบเหล่านี้ดี Rogue Core อาจกลายเป็นอีก pillar หนึ่งของ Ghost Ship ไม่ใช่แค่ side project

## มุมมองของผม

ผมมองว่า Rogue Core เป็นข่าวเกมที่น่าจับตา เพราะมันเดินบนเส้นบาง ๆ ระหว่าง "ใช้ IP เดิมให้คุ้ม" กับ "กล้าทดลอง loop ใหม่"

ในตลาดที่ live-service fatigue สูงขึ้น เกมที่ยอมเริ่มจาก Early Access แบบชัดเจน และเปิดให้ community ช่วยจูนตั้งแต่ต้น อาจมีโอกาสสร้างความเชื่อใจได้มากกว่าเกมที่ประกาศตัวเต็มแล้วค่อยแก้ทีหลัง

สรุปสั้น ๆ:
**Deep Rock Galactic: Rogue Core คือบททดสอบว่าแฟรนไชส์ co-op ที่มี community แข็งแรงจะขยายไปสู่ roguelite standalone ได้แน่นแค่ไหน**
