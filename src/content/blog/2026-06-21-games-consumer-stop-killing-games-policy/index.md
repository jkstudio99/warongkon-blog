ข่าว **Games / Consumer** สำหรับรอบวันที่ **21 มิถุนายน 2026** คือประเด็น **Stop Killing Games** กลับมาอยู่ในกระแสหลัก หลัง The Guardian เผยแพร่รายงานวันที่ **19 มิถุนายน 2026** ว่าการเคลื่อนไหวเพื่อหยุดการ "ทำลาย" เกมออนไลน์หลังปิด server กำลังเข้าสู่สนามกฎหมายทั้งในยุโรปและรัฐ California

เรื่องนี้ไม่ใช่แค่ nostalgia ของคนเล่นเกมเก่า แต่เป็นคำถาม consumer rights ที่จริงจังขึ้นมาก: ถ้าผู้เล่นซื้อเกมมาแล้ว publisher มีสิทธิ์ปิด server จนเกมใช้งานไม่ได้ทั้งหมดหรือไม่ และควรมีกติกา end-of-life plan อย่างไร

## จุดเริ่มจาก The Crew และคำถามเรื่อง ownership

แคมเปญ Stop Killing Games เริ่มถูกพูดถึงหนักหลังกรณี **The Crew** ของ Ubisoft ที่ปิดบริการและทำให้เกมแข่งรถออนไลน์ที่ผู้เล่นเคยซื้อไม่สามารถเล่นได้ตามเดิม

เว็บไซต์ทางการของ Stop Killing Games ระบุว่าเป้าหมายของแคมเปญคือผลักดันให้ publisher ไม่สามารถทำให้เกมที่ขายไปแล้วใช้งานไม่ได้อย่างถาวรโดยไม่มีทางเลือกใด ๆ ให้ผู้บริโภค

คำถามหลักไม่ใช่ว่า publisher ต้องให้บริการ server ไปตลอดกาลหรือไม่ เพราะต้นทุนของ live service มีจริง แต่คำถามคือเมื่อถึงวันปิดบริการ publisher ควรต้องมีทางเลือก เช่น offline mode, private server, server binaries, patch หรือ refund ในบางกรณีหรือไม่

สำหรับเกมยุคใหม่ที่พึ่งพา authentication, matchmaking, cloud save และ live economy มากขึ้นเรื่อย ๆ เส้นแบ่งระหว่าง "ซื้อเกม" กับ "เช่าสิทธิ์เข้าถึงบริการ" เริ่มเบลอจนผู้บริโภคจำนวนมากรู้สึกว่ากติกาเดิมไม่พอ

## European Commission ตอบ ECI วันที่ 16 มิถุนายน 2026

เอกสารของ European Commission ลงวันที่ **16 มิถุนายน 2026** เกี่ยวกับ European Citizens' Initiative ชื่อ **"Stop Destroying Videogames"** ยืนยันว่าแคมเปญถูกยื่นเข้าสู่กระบวนการหลังผ่าน threshold ที่กำหนด

คณะกรรมาธิการอธิบายปัญหาว่าเกมบางประเภทอาจใช้งานไม่ได้เมื่อ publisher ปิดบริการ online server และรับทราบข้อเสนอของผู้จัดแคมเปญที่ต้องการให้ผู้บริโภคสามารถเล่นเกมต่อได้ในระดับสมเหตุสมผลหลังการสนับสนุนทางการสิ้นสุด

แต่คำตอบของ Commission ก็สะท้อนความซับซ้อน: การบังคับให้ publisher เปิด server code หรือ asset บางอย่างอาจชนกับ copyright, cybersecurity, personal data และ business model ของเกมออนไลน์

พูดง่าย ๆ คือ EU ยังไม่ได้บอกว่า publisher ต้องทำอะไรทันที แต่ยอมรับว่าปัญหานี้เป็น policy issue ที่ต้องพิจารณาจริง ไม่ใช่แค่เสียงบ่นในชุมชนเกม

## California AB 1921 ทำให้เรื่องนี้ใกล้กฎหมายจริงขึ้น

ฝั่งสหรัฐฯ ประเด็นนี้เดินหน้าผ่านร่างกฎหมาย **California AB 1921** หรือ **Protect Our Games Act**

ข้อความร่างกฎหมายระบุว่าเกมที่วางจำหน่ายหรือ re-release ตั้งแต่วันที่ **1 มกราคม 2027** เป็นต้นไปจะต้องมีเงื่อนไขบางอย่างหากผู้จัดจำหน่ายจะปิดบริการที่ทำให้เกมใช้งานไม่ได้ เช่น แจ้งล่วงหน้า 60 วัน, จัดหา alternative version, patch/update หรือ refund ในเงื่อนไขที่กำหนด

The Guardian รายงานว่าร่างกฎหมายผ่าน Assembly แล้วและจะไปต่อที่ Senate ซึ่งทำให้ประเด็น game preservation ขยับจาก campaign สู่ legislative process ที่จับต้องได้มากขึ้น

แน่นอนว่าฝั่ง industry ไม่ได้เห็นด้วยทั้งหมด Entertainment Software Association เผยแพร่ความเห็นคัดค้าน โดยมองว่าร่างกฎหมายนี้อาจทำให้ต้นทุนสูงขึ้น กระทบ developer และชนกับข้อจำกัดด้าน copyright กับการดำเนินงานของเกม live service

## เกมออนไลน์ไม่ใช่ media แบบเดิม

ข่าวนี้น่าสนใจเพราะเกมออนไลน์เป็น media ที่มีคุณสมบัติไม่เหมือนหนัง เพลง หรือหนังสือ

ผู้เล่นไม่ได้ซื้อแค่ไฟล์ แต่ซื้อประสบการณ์ interactive ที่มักต้องอาศัย server, account, economy และชุมชน เมื่อ server หยุด เกมอาจไม่ได้แค่ "หมด support" แต่หายไปจากการใช้งานจริง

ฝั่ง publisher มีเหตุผลเรื่องต้นทุนและความปลอดภัย แต่ฝั่งผู้บริโภคก็มีเหตุผลเรื่อง ownership และ preservation โดยเฉพาะเมื่อเกมถูกขายในราคาเต็ม หรือถูกทำตลาดด้วยภาษาที่ทำให้ผู้บริโภครู้สึกว่าได้ซื้อสินค้า ไม่ใช่เช่าการเข้าถึงแบบไม่มีกำหนดชัด

นี่คือ tension ที่กฎหมาย consumer protection แบบเดิมยังตอบได้ไม่เต็ม

## มุมมองของผม

ผมมองว่า Stop Killing Games คือการเปลี่ยนบทสนทนาของ industry จาก "เกมเป็นบริการ" ไปสู่ "บริการต้องมีแผนจบชีวิตที่เป็นธรรม"

publisher ไม่จำเป็นต้องถูกบังคับให้แบก server ทุกเกมตลอดไป แต่ควรถูกถามอย่างจริงจังว่าเมื่อขายเกมให้ผู้บริโภคแล้ว วันปิดบริการควรเหลืออะไรไว้บ้าง

สิ่งที่น่าจับตาคือกฎหมายจะเขียนแคบหรือกว้างแค่ไหน ถ้ากว้างเกินไปอาจทำให้ developer เล็กกลัวการทำ online features แต่ถ้าแคบเกินไปก็ไม่แก้ปัญหาเกมที่หายไปจริง

สรุปสั้น ๆ:
**วันที่ 19 มิถุนายน 2026 The Guardian ชี้ว่า Stop Killing Games กำลังขยับเข้าสู่สนาม policy จริง โดยมีทั้งคำตอบจาก European Commission วันที่ 16 มิถุนายน และร่าง California AB 1921 ที่ตั้งเป้าเกมใหม่ตั้งแต่ 1 มกราคม 2027 ข่าวนี้ทำให้ game preservation กลายเป็น consumer rights issue อย่างเต็มตัว**
